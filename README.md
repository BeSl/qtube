##Необходимое ПО:

- node.js
- PostgreeSQL

## Структура проекта


`   -src-
- backend - сервис для работы с СУБД
- fileserv - сервис для доступа к файлам
- frontend - интерфейс пользователя
`
из каждого каталога
# установить необходимые пакеты

```
    npm i
```

# настроить .env

# запуск
```
    npm start //back, front
    node node.js
```
